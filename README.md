# Daily GitHub Contributions

**Просто загрузите эту папку на GitHub один раз — и всё!**

Каждый день автоматически будет появляться зелёная клетка в вашем профиле GitHub.

## Что делать:

1. Создайте новый репозиторий на GitHub
2. Загрузите всю эту папку в репозиторий
3. В настройках репозитория: **Settings → Actions → General → Workflow permissions → Read and write permissions**
4. Готово!

После этого GitHub Actions будет автоматически делать коммиты каждый день. Больше ничего делать не нужно.
